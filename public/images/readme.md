# Thư mục ảnh duy nhất của FitBlend

Mọi hình ảnh dự án đặt tại đây. Code tham chiếu qua `src/app/config/images.ts`.

## Cấu trúc

| File / thư mục | Mục đích |
|----------------|----------|
| `fitblend_hero_smoothie.png` | Ly smoothie — landing, sản phẩm Bơ/Matcha |
| `fitblend_combo_bottles.png` | Combo/chai — landing, section mua sỉ |
| `strawberry_smoothie.png` | Ảnh nhóm vị dâu |
| `mango_smoothie.png` | Ảnh nhóm vị xoài/nhiệt đới |
| `cacao_oat_smoothie.png` | Ảnh nhóm cacao/cà phê |
| `menu1.jpg`, `menu2.jpg` | Bảng giá popup landing |
| `pos-icon.svg` | Icon PWA màn POS |
| `uploads/` | Ảnh admin upload qua Admin → Sản phẩm |

Thay ảnh: copy file mới **giữ nguyên tên** hoặc cập nhật đường dẫn trong `src/app/config/images.ts`.
